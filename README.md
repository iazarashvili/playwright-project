# ­ЪјГ Playwright Test Suite

This project uses [Playwright](https://playwright.dev/) to perform end-to-end testing across modern browsers.

## ­ЪЊд рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљрЃЏрЃЊрЃћ рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃАрЃљрЃГрЃўрЃарЃЮ рЃърЃљрЃЎрЃћрЃбрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ:

```bash
npm install
npx playwright install
npx playwright test
```

## РџЎ№ИЈ рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ рЃЊрЃљ рЃЏрЃљрЃЌрЃў рЃљрЃдрЃгрЃћрЃарЃљ

| рЃЉрЃарЃФрЃљрЃюрЃћрЃЉрЃљ | рЃљрЃдрЃгрЃћрЃарЃљ |
|----------|--------|
| `npx playwright test` | рЃДрЃЋрЃћрЃџрЃљ end-to-end рЃбрЃћрЃАрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ. |
| `npx playwright test --ui` | рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў UI рЃарЃћрЃЪрЃўрЃЏрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃбрЃћрЃАрЃбрЃўрЃарЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃћрЃЉрЃљрЃњрЃўрЃюрЃњрЃўрЃАрЃЌрЃЋрЃўрЃА. |
| `npx playwright test --project=chromium` | рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ Chromium рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃќрЃћ (рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, Google Chrome). |
| `npx playwright test example` | рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃцрЃљрЃўрЃџрЃўрЃА (рЃЏрЃљрЃњ. `example.spec.ts`) рЃбрЃћрЃАрЃбрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ. |
| `npx playwright test --debug` | рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ рЃЊрЃћрЃЉрЃљрЃњрЃўрЃА рЃарЃћрЃЪрЃўрЃЏрЃерЃў, Playwright Inspector-рЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ. |
| `npx playwright codegen` | рЃбрЃћрЃАрЃбрЃћрЃЉрЃўрЃА рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃў рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃўрЃА рЃЕрЃљрЃюрЃљрЃгрЃћрЃарЃўрЃА рЃАрЃљрЃцрЃБрЃФрЃЋрЃћрЃџрЃќрЃћ. |
